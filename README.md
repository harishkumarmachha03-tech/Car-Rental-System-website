# Car-Rental-System-website

A responsive Car Rental System Website developed using HTML and CSS, designed to provide users with a smooth and interactive experience for browsing, booking, and renting cars online.

📌 Project Description

This project is a frontend-based car rental platform where users can explore different cars, proceed with booking, make payments, and view confirmation details. It simulates a real-world car rental workflow with multiple pages and user interactions.

✨ Features
🚘 View available cars with images
📄 Dedicated pages for car listings (cars.html)
📅 Booking system (booking.html)
💳 Payment page (payment.html)
✅ Payment status & confirmation pages
📩 Confirmation screen after successful booking
📱 Responsive design using CSS
🎨 Clean UI with attractive backgrounds and images

🚀 How to Run the Project

Download or clone the repository:

git clone https://github.com/your-username/car-rental-system.git
Open the project folder

Run the project by opening:

index.html

in any web browser

🔄 Workflow of the Website
User opens Home Page
Navigates to Cars Page
Selects a car and proceeds to Booking Page
Enters details and goes to Payment Page
Completes payment → sees Payment Status
Gets Confirmation Page
